### What I Learned

In this project, I focused on improving my understanding of:

* BEM methodology for structuring CSS
* Writing semantic HTML
* Local hosting of fonts

I also identified areas where I can improve. For example, I initially approached the layout using a desktop-first strategy, but this project would have likely been easier to build using a mobile-first workflow. Additionally, I want to strengthen my understanding of media queries for responsive design.

---

### Continued Development

In future projects, I plan to:

* Improve my responsive design skills using media queries
* Practice and apply the SMACSS methodology
* Refine my overall CSS organization and scalability
